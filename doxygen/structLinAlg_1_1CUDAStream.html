Detailed Description

Class to encapsulate a CUDA stream.

The stream is bound to a device and includes a cublasHandle and cusparseHandle associated with the stream. The stream is ordered but can't be synchronized on arbitrary positions. It only supports CUDA, cuBLAS and cuSPARSE functions.
