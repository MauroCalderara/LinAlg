Detailed Description

MPIStream.

The MPI stream handle allows for synchronization of multiple asynchronous MPI calls. However, there is no ordering guarantee for the calls (as there is none with one synchronization unit in MPI itself). It is in that sense not appendable as it does not guarantee that earlier operations are completed before later operations.
